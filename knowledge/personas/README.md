# Personas

This folder stores your personas for synthetic user testing.

**No pre-built personas are included** — you create your own through the `/persona-builder` skill or during a test session. Saved personas appear here automatically and persist across sessions.

## How personas get here

1. **During a test session** — when you create a persona from scratch, you'll be asked "Save for future sessions?" → saved here
2. **Via `/persona-builder`** — dedicated persona creation skill → saved here
3. **Manually** — drop any markdown file here following the persona format

## Managing personas

- **List:** say "show my personas" in any session
- **Delete:** say "delete [name]" — confirmation required before deletion
- **All personas here are yours** — no protected/pre-built files

## File naming

Saved personas use the format: `custom-[name-slug].md` (e.g., `custom-marcus-rivera.md`)
